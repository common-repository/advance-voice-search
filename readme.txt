A plugin to add voice search functionality to your WordPress site.

== Description ==

Advance Voice Search is a powerful and easy-to-use plugin that adds voice search capabilities to your WordPress website. Enhance user experience by allowing visitors to search your site using voice commands. This plugin integrates seamlessly with your existing search functionality and supports multiple languages.

Right now, this plugin only supports browsers that are compatible with 'SpeechRecognition'. For full details, See this [technical reference](https://developer.mozilla.org/en-US/docs/Web/API/Web_Speech_API) for more details.

**Features:**
* Simple setup and integration
* Supports multiple languages
* Customizable interface
* Compatible with most WordPress themes

== Installation ==

1. Upload the `advance-voice-search` folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Visit your website and start searching using your voice.

== Frequently Asked Questions ==

= How do I configure the plugin? =

Once the plugin is activated, go to the plugin settings page in your WordPress dashboard to configure the voice search options.

= Does this plugin support multiple languages? =

Yes, Advance Voice Search supports multiple languages.

= Will this plugin work with any WordPress theme? =

Advance Voice Search is designed to be compatible with most WordPress themes. If you encounter any issues, please contact our support.
